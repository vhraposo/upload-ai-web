# 💻 Projeto
Aplicação que possibilita realizar upload de vídeos e, por meio de IA, criar automaticamente títulos chamativos e descrições com uma boa indexação.

- Radix Ui
- Axios
- Tailwind
- lib: ffmpeg

- OpenAI API
